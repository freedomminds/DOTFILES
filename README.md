### Cara Install rcm di UBUNTU / Debian

```bash
sudo apt install rcm
```

Cara Memasukan file ke dotfiles

```bash
mkrc .vimrc
```

untuk mengecek sebuah file yang suda di link ke dotfile

```bash
ls -al
```

atau

```bash
lsrc
```

Cara agar configurasi kita bisa kita gunakan lagi

```bash
rcup
```
